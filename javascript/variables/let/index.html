let — escopo real e fim das surpresas

1. Declaração com let

let nome = "Sanciweferson"
Resultado →
A criação é simples. O diferencial aparece quando o código cresce.

2. Redeclaração bloqueada

let nome = "San"
let nome = "Outro"
Resultado →
Redeclarar no mesmo escopo gera erro. Bug eliminado cedo.

3. Reatribuição permitida

nome = "Weferson"
Resultado →
let permite mudar o valor. Controle sem rigidez.

4. Hoisting com let (TDZ)

console.log(x)
let x = 10
Resultado →
A variável existe, mas está na Temporal Dead Zone.

5. Escopo de bloco

if (true) {
  let idade = 20
}
console.log(idade)
Resultado →
Blocos agora são escopos reais. Sem vazamentos.

6. let no loop

for (let i = 0; i < 1; i++) {}
console.log(i)
Resultado →
O índice morre com o loop. Escopo limpo.

7. Closure correta com let

let funcoes = []

for (let i = 0; i < 3; i++) {
  funcoes.push(() => i)
}

f[1]() →
f[2]() →
Cada iteração cria um i novo. Closure funciona sem truque.
